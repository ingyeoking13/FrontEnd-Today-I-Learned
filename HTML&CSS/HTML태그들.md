# HTML태그들

## Heading Tag

문서 내에 제목을 표현할 때 사용되는 태그이다. 

```html
<h1></h1>
<h2></h2>
...
<h5></h5>
<h6></h6>
```

## Paragraph Tag

`<p></p>`는 `단락 태그`이며 문단을 만들 때 사용된다. 

## linebreak Tag 

`<br></br>`는 `개행 태그`이며 문장 간의 개행을 위해 사용된다. *HTML에서 내용은 하나 이상의 공백 외엔 공백 및 개행이 허용 되지 않기 때문에, 개행을 위해서라면 반드시 `<br></br>` 태그를 이용하여야 한다.*
